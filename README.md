Problem Statement: Many students struggle to understand complicated educational documents such as:
1.Scholarship eligibility rules
2.Government guidelines
3.Exam instructions
4.Admission procedures
5.Official notifications
These documents are often written in complex legal and technical language, making them hard to interpret.
EduAI solves this by converting difficult educational text into easy-to-understand, plain language, while also providing AI chat support, document reading, reminders, and pathways for students.

Tech stack: 
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

Setup + Installation steps: 
1. Clone the repository
2. Install dependencies
3. Start the development server
4. Open in browser

Features: Plain Language Translator Converts complex scholarship/exam rules into easy conversational text.
1.Multilingual AI Chat : Supports English + Indian languages for student queries.

2. Document Reader: Upload PDFs / text files and convert content to plain language.

3.Text-to-Speech: Reads difficult content aloud for students.

4. Career Pathway Finder: Generates academic and career roadmap based on user choices.

5. Reminders System :Add reminders for deadlines (scholarships, exams, forms).

6. Clean and Responsive UI: Built with shadcn components and TailwindCSS.
